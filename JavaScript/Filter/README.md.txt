Рост учеников

Учебный проект на JavaScript, направленный на отработку работы с массивами, фильтрацией данных и обновлением DOM-элементов.
Приложение отображает список чисел, обозначающих рост учеников, и позволяет добавлять новые значения, а также фильтровать список по минимальному росту.

Цель задания

Научиться выполнять фильтрацию чисел в массиве и показывать обновлённый результат в списке DOM-элементов, закрепить навыки работы с циклами, условиями и взаимодействием с пользователем через prompt() и alert().

Функционал приложения

-Отображение списка роста учеников из массива на странице
-Добавление нового значения через кнопку «Добавить рост»
-Ввод осуществляется в окне prompt()
-Если пользователь ничего не ввёл, отображается сообщение alert("Рост не введён!")
-Фильтрация списка по минимальному значению через кнопку «Фильтровать»
-Пользователь вводит число для фильтрации
-Отображаются только значения, больше или равные введённому числу
-Если пользователь ничего не ввёл, показывается весь список без фильтрации
-Минимальная CSS-стилизация для визуальной наглядности

Что отработано

-Создание и работа с массивами чисел
-Использование методов filter(), push() и перебора элементов
-Динамическое обновление списка элементов в DOM
-Проверка пользовательского ввода и вывод сообщений через alert()
-Работа с prompt() для получения данных от пользователя
-Базовая перерисовка контента при изменении массива
