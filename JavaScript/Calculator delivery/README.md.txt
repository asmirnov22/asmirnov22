Калькуляция стоимости доставки товаров

Учебный проект на HTML, CSS и JavaScript, в котором реализована веб-страница для расчёта стоимости доставки товаров на основе введённых пользователем данных.
Форма позволяет добавлять несколько товаров, рассчитывать стоимость каждого из них и отображать итоговый список в виде таблицы.

Цель задания

Научиться работать с пользовательскими формами и выполнять валидацию введённых данных, а также производить вычисления и отображать результаты в DOM.
Проект помогает закрепить навыки работы с HTML-формами, циклами и динамическим созданием элементов на странице.

Функционал приложения

Пользователь может:

-Ввести название товара
-Указать вес товара (в килограммах)
-Указать расстояние доставки (в километрах)
-Добавить товар в список для расчёта

После добавления каждый товар отображается в таблице со следующими данными:

-Название товара
-Вес (кг)
-Расстояние (км)

Стоимость доставки, рассчитанная по формуле:

стоимость = (вес * расстояние) / 10

Валидация данных

Перед добавлением товара выполняется проверка:

-Все поля заполнены
-Вес и расстояние — положительные числа
-Название товара содержит только буквы и пробелы

Если данные введены некорректно, пользователю отображается alert с подсказкой об ошибке.

Что отработано

-Создание и обработка HTML-форм
-Валидация пользовательских данных на JavaScript
-Выполнение арифметических вычислений
-Работа с DOM: добавление строк в таблицу
-Очистка формы после успешного добавления
-Минимальная стилизация интерфейса
