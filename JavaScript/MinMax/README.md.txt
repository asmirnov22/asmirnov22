Поиск минимального и максимального числа

Небольшой учебный проект на JavaScript, демонстрирующий работу с массивами и циклами.
Программа находит минимальное и максимальное значения в заданном массиве чисел и выводит результат на экран.

Что отработано

-Создание и работа с массивами в JavaScript
-Использование циклов for для перебора элементов массива
-Сравнение числовых значений и поиск минимума / максимума
-Динамическое изменение содержимого страницы с помощью DOM (textContent)
-Работа с элементами через document.querySelector
