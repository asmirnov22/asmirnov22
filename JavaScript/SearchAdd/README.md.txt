Домашняя библиотека

Учебный проект на JavaScript, направленный на отработку навыков работы с массивами, поиском элементов и взаимодействием с DOM.
Приложение позволяет просматривать список книг, добавлять новые и выполнять поиск по названию.

Цель задания

Научиться выполнять поиск элементов в массиве и отображать результат в списке DOM-элементов, а также закрепить навыки работы с функциями, циклами и обработкой пользовательского ввода.

Функционал приложения

-Отображение списка книг из заранее заданного массива
-Добавление новой книги через окно prompt()
-Если пользователь не ввёл название — отображается предупреждение alert("Название книги не введено!")
-Поиск книги по названию через кнопку «Поиск»
-Если книга найдена — она подсвечивается цветом
-Если книга не найдена — выводится сообщение alert("Книга не найдена!")
-Минимальная CSS-стилизация для наглядности

Что отработано в проекте

-Создание и работа с массивами
-Добавление и поиск элементов в массиве
-Манипуляции с DOM: создание, обновление и выделение элементов списка
-Работа с пользовательским вводом через prompt() и alert()
-Проверка корректности данных перед добавлением
-Применение базовых CSS-стилей
