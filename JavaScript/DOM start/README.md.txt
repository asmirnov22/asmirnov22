Каталог товаров

Небольшой учебный проект на JavaScript, демонстрирующий базовую работу с DOM и обработку событий.
При нажатии на кнопку рядом с товаром — он добавляется в список корзины.

Описание проекта

На странице отображается список товаров с кнопками «Добавить в корзину».
Каждый клик по кнопке вызывает JavaScript-функцию addToCart(), которая:
Создаёт новый элемент списка (<li>)
Добавляет в него название выбранного товара
Помещает этот элемент в раздел корзины (<ul id="cart">)
Таким образом реализуется простое динамическое обновление страницы без перезагрузки.

Что отработано

-Работа с DOM-элементами через document.querySelector и createElement
-Добавление элементов в HTML при помощи append()
-Обработка кликов через встроенный атрибут onclick
-Изменение содержимого страницы с помощью JavaScript

Используемые технологии

HTML5 — структура страницы
CSS (опционально) — оформление (если добавляется)
JavaScript (ES6) — логика добавления товаров
