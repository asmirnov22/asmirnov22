Форма опроса пользователя

Учебный проект на HTML, CSS и JavaScript, в котором реализована веб-форма для сбора информации о пользователе.
Форма включает несколько типов полей ввода, проверку данных и вывод введённой информации на экран после успешной отправки.

Цель задания

Научиться создавать интерактивные веб-формы и выполнять валидацию пользовательских данных на стороне клиента с помощью JavaScript.
Закрепить навыки работы с элементами формы, событиями (input, submit) и отображением данных в DOM.

Функционал приложения

Форма собирает следующие данные:

-Имя пользователя — текстовое поле (обязательно для заполнения).
-Email — текстовое поле с проверкой на правильный формат email.
-Пол — выбор между «Мужской» и «Женский» с помощью радиокнопок.
-Оценка сервиса — ползунок (range input) от 1 до 10.

Интересы пользователя — несколько вариантов для выбора (checkbox):

-Спорт
-Музыка
-Путешествия
-Кино

Дополнительные комментарии — текстовое поле для произвольного ввода.

Кнопка “Отправить” — при нажатии происходит проверка всех обязательных полей и отображение введённой информации на странице.

Что отработано

-Создание форм с использованием разных типов input-элементов
-Работа с событиями input и submit
-Валидация данных (проверка имени и email)
-Получение и обработка данных из элементов формы
-Вывод собранной информации в DOM
-Минимальная стилизация формы с помощью CSS
