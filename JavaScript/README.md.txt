В этой папке собраны мои учебные проекты, выполненные в рамках курса
«Профессия Фулстэк-разработчик на JavaScript 3.0» от Skillbox.

Каждый проект направлен на закрепление определённых навыков фронтенд-разработки (а в будущем и бэкенд):
от базового JavaScript до работы с DOM.

Цели и задачи

-Закрепление ключевых навыков JavaScript
-Отработка работы с DOM, событиями, циклами, массивами и функциями
-Практика написания чистого, читаемого и модульного кода

Используемые технологии

-HTML5 / CSS3 / JavaScript (ES6+)
-Flexbox / Grid
-БЭМ

Курс Skillbox: Профессия Фулстэк-разработчик на JavaScript 3.0 включает комплексную подготовку по фронтенд- и бэкенд-направлению, охватывая HTML, CSS, JS, работу с фреймворками и серверной частью.
Эти проекты являются частью практической программы и демонстрируют мой прогресс в освоении JavaScript.
