Учебные проекты по веб-разработке

Добро пожаловать в мою коллекцию учебных проектов по веб-разработке.
Здесь собраны работы, выполненные в процессе изучения HTML, CSS и JavaScript — от базовой вёрстки до динамических интерактивных приложений.

Структура репозитория
 HTML & CSS

В этой папке находятся проекты, посвящённые основам вёрстки и стилизации веб-страниц.
Каждая работа направлена на отработку отдельных приёмов:

работа с сетками и адаптивностью;
применение методологии БЭМ;
настройка локальных шрифтов;
псевдоклассы и интерактивные элементы без JavaScript.

 JavaScript

Здесь собраны проекты, демонстрирующие работу с DOM, формами, массивами, циклами и обработкой данных.
В проектах реализованы:

-динамическое добавление элементов в DOM;
-работа с пользовательскими формами и валидацией;
-поиск, фильтрация и сортировка данных;
-простая логика калькуляции и отображения результатов на странице.

Цель

Постепенно освоить путь от статических страниц к интерактивным веб-приложениям, развивая навыки структурирования кода, работы с событиями, логикой и интерфейсом.

План развития

-Добавление новых проектов и мини-приложений
-Усложнение логики и структуры кода
-Освоение работы с API и JSON
-Переход к фреймворкам (React, Vue)

Итог

Этот репозиторий — мой личный путь обучения веб-разработке.
С каждым новым проектом код становится чище, структурнее и функциональнее, а интерфейсы — удобнее и современнее.
