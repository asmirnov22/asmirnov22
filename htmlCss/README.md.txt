Учебные проекты по HTML и CSS

В этой папке собраны мои учебные проекты, выполненные в процессе изучения вёрстки сайтов с нуля.
Каждый проект направлен на отработку навыков семантической HTML-разметки, стилизации с помощью CSS,
а также освоение адаптивной вёрстки и базовых интерактивных эффектов.А в будущем эта папка будет пополнятся более сложными и интересными работами

Цели и задачи

-Освоить чистую и семантическую HTML-разметку
-Научиться использовать БЭМ для структурирования кода
-Отработать Flexbox и Grid Layout
-Реализовать адаптивную вёрстку под разные устройства
-Работа с псевдоэлементами
-Работа с формами

Используемые технологии

-HTML5
-CSS3
-Flexbox / Grid
-БЭМ (Блок-Элемент-Модификатор)
-Адаптивная вёрстка (Responsive Design)
-Google Fonts / Локальные шрифты

О проектах

Эти проекты были созданы для практики и пополнения портфолио.
Они демонстрируют умение работать с макетами из Figma, аккуратно вёрстать интерфейсы и соблюдать стандарты HTML и CSS.

Каждый проект находится в отдельной папке с собственным README.md.
Чтобы просмотреть — просто откройте файл index.html в браузере или запустить через локальный сервер.
