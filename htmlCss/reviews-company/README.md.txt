TechZone — Сетка отзывов

Учебный проект по верстке сетки и карточек отзывов для сайта TechZone, специализирующегося на продаже компьютерной техники.  
Проект выполнен по макету из Figma с применением методологии **БЭМ** и локальным подключением шрифтов.


 Используемые технологии

-HTML5
-CSS3
-БЭМ
-Flexbox
-Локальное подключение шрифтов

  Что я отработал в этом проекте

- Применение БЭМ для структурированной и масштабируемой верстки  
- Настройку локальных шрифтов  
- Создание адаптивной сетки отзывов с помощью Flexbox  
- Точечное позиционирование элементов по макету  



Проект создан в рамках курса **Skillbox: Профессия Фулстэк-разработчик на JavaScript 3.0**.
