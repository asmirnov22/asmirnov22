Maker — одностраничный сайт

Проект представляет собой одностраничный сайт (лендинг), выполненный по макету, найденному в открытом доступе в Figma.
Сайт создан для демонстрации продуктов компании, специализирующейся на рекламных услугах.

Вёрстка выполнена самостоятельно, без использования шаблонов и сторонних фреймворков.

Используемые технологии

HTML5
CSS3
Flexbox
БЭМ (Блок-Элемент-Модификатор)
Локальное подключение шрифтов

Что было реализовано

Структурированная и масштабируемая вёрстка по методологии БЭМ
Адаптивная сетка отзывов с использованием Flexbox
Точное позиционирование элементов по макету
Настройка локальных шрифтов
Реализация hover-эффектов для интерактивных элементов
Корректная работа кнопок с изменением курсора при наведении
Отлаженные отступы и визуальные пропорции для разных экранов

О проекте

Проект разработан как часть портфолио для демонстрации навыков вёрстки и адаптивного дизайна.
Цель — показать умение работать с макетами, реализовывать чистый, аккуратный код и визуально точные интерфейсы.
