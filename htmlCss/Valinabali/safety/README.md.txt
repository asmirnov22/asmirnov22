Valinabali — Страница безопасности

Учебный проект по верстке страницы «Безопасность» для туристического сайта Valinabali.
Основной упор сделан на создание адаптивной сетки с помощью CSS Grid и аккуратное оформление информационных блоков по макету из Figma.


Используемые технологии

-HTML5
-CSS3
-CSS Grid Layout
-SVG-спрайты
-Flexbox (для внутренних элементов)
-БЭМ
-Адаптивная верстка


Что я отработал в этом проекте

-Создание гибкой сетки блоков с применением Grid Layout
-Отработка БЭМ-наименований для структурной верстки
-Создание адаптивных отступов
-Практика по точному соответствию макету из Figma
-Подключение и использование локальных шрифтов


Проект создан в рамках курса **Skillbox: Профессия Фулстэк-разработчик на JavaScript 3.0**.
