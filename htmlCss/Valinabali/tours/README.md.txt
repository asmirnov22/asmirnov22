Valinabali — Сетка туров

Учебный проект по верстке страницы с турами для сайта Valinabali, посвящённого путешествиям и отдыху на Бали.
Проект выполнен по макету из Figma с применением CSS Grid, локальным подключением шрифтов и использованием SVG-спрайтов.


Используемые технологии

-HTML5
-CSS3
-CSS Grid Layout
-SVG-спрайты
-Flexbox (для внутренних элементов)
-БЭМ
-Адаптивная верстка


Что я отработал в этом проекте

-Создание нестандартной сетки карточек с помощью CSS Grid
-Позиционирование карточек в разных ячейках сетки по макету
-Работа с SVG-спрайтами для иконок и декоративных элементов
-Подключение локальных шрифтов
-Работа с макетом Figma — точное повторение сетки и типографики


Проект создан в рамках курса **Skillbox: Профессия Фулстэк-разработчик на JavaScript 3.0**.
