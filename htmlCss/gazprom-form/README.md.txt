Бизнес Онлайн — Страница входа и регистрации

Учебный проект по верстке страниц для продукта «Газпромбанк: Бизнес Онлайн», включающий страницу входа, вариацию с окном восстановления пароля и страницу регистрации.
Верстка выполнена по макету из Figma с применением методологии БЭМ и современных инструментов CSS.

Используемые технологии

-HTML5
-CSS3
-БЭМ
-Flexbox
-Grid
-Локальное подключение шрифтов

Что я отработал в этом проекте

-Создание форм входа, регистрации и восстановления пароля
-Применение БЭМ для структурированной и масштабируемой верстки
-Работа с UI-kit: поля ввода (input), кнопки, состояния элементов
-Изучение и использование псевдоклассов (:hover, :focus, :active)
-Настройку адаптивного интерфейса и точное позиционирование элементов по макету
-Подключение и настройку локальных шрифтов

Проект выполнен в рамках курса Skillbox: Профессия Фулстэк-разработчик на JavaScript 3.0.
